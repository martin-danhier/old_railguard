Detailed Description

A "shader effect" is a configuration of pipeline layouts, shaders and descriptor set. Materials can use several shader effects (e.g. for forward and shadow rendering), and a single shader effect can be used by several pipelines (for example a "default lit" shader effect).

Create a Shader Effect and add it to the vectors. The effect will not be built yet, meaning the pipeline will be null. Call BuildEffect() to build it, or set the "build" parameter to true.
